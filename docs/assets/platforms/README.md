# Platform icon sources

- `youtube.svg` and `bilibili.svg` use paths from [Simple Icons](https://simpleicons.org/), CC0-1.0.
- `douyin.svg` uses the Simple Icons TikTok glyph as the closest available recognition mark, recolored and retitled for the Douyin row.
- `local-file.svg` and `direct-url.svg` are project-authored generic icons.

Platform names and marks belong to their respective owners. They are shown only to identify tested input sources and do not imply endorsement, partnership, or an official integration.
